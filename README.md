# BuiltforME

A personal, low-impact strength plan — the gym plan for people who'd rather not.

A single self-contained `index.html` (styles, images, and JavaScript inlined). Open it in any browser; on iPhone, use Safari → Share → **Add to Home Screen** for an app-like shortcut.

## Features
- 5-day weekly plan with per-set checkboxes that persist on your device.
- A training calendar to log the days you actually trained, with this-week / this-month / all-time counters.
- Dark-mode aware, mobile-first layout.
